Para el correcto funcionamiento de este WebService se debe configurar el archivo "application.properties" 
que se encuentra en la siguente ruta:
".../src/main/resources"

Content "application.properties"

spring.datasource.url=jdbc:mysql://localhost:3306/test
spring.datasource.username=admin
spring.datasource.password=admin
spring.jpa.hibernate.ddl-auto=update  // "update" si se quiere crear automaticamente las tablas, "none" si las tablas ya estan creadas

Todas las pruebas de solicitudes (POST/GET) se realizaron con Postman y se trabajo con una base de datos MariaDB

Estructura Json Rut

{
    "rut": "20107983",
    "verifyCode" : "7"
}
